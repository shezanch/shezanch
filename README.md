## Featured project

### StratifyAI

StratifyAI is an AI-powered content strategy tool built with Python, Streamlit, BeautifulSoup, and the Gemini API.

It helps users:

* Find trending topics
* Analyze website content
* Generate content ideas for Instagram, TikTok, and LinkedIn
* Create platform-specific content calendars
